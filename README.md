# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Flask, NLP, and Sentence Transformers.

## Features

- Upload Resume (PDF)
- Resume Skill Extraction
- Resume Similarity Score
- AI-Based Resume Analysis
- User-Friendly Web Interface

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Sentence Transformers
- NLP

## Project Structure

│

## Installation

```bash
git clone https://github.com/Vanshu27-tech/AI-Resume-Analyzer.git
```

```bash
cd AI-Resume-Analyzer
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```
