# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python**, **Flask**, **Natural Language Processing (NLP)**, and **Sentence Transformers**. The application allows users to upload both a **Resume** and a **Job Description (JD)** to perform intelligent analysis and determine how well the resume matches the job requirements.

---

## 🚀 Features

- 📄 Upload Resume (PDF)
- 📝 Upload Job Description (JD)
- 🤖 AI-powered Resume Analysis
- 🎯 Resume & Job Description Matching
- 📊 Resume Similarity Score
- 🔍 Skill Extraction
- 💡 AI-based Resume Improvement Suggestions
- ⚡ Fast and User-Friendly Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Vanshu27-tech/AI-Resume-Analyzer.git
```

### Open the Project Folder

```bash
cd AI-Resume-Analyzer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

---

## 💡 How It Works

1. Upload your Resume (PDF).
2. Upload the Job Description (JD).
3. The AI extracts important skills and keywords.
4. The system compares the Resume with the Job Description.
5. A similarity score and personalized recommendations are generated.
